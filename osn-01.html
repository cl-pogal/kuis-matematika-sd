1
Number Theory / Teori Bilangan
a)
Digit 81: \(8+1=9\). Cek: \(\frac{1}{9} \times 81 = 9\). Jumlah digit = \(\frac{1}{9}\) dari bilangan. ✓
Digits of 81: \(8+1=9\). Check: \(\frac{1}{9}\times81=9\). Sum of digits = \(\frac{1}{9}\) of the number. ✓
Terbukti / Proven ✓
b)
Misalkan bilangan dua digit = \(\overline{ab} = 10a+b\), syarat: \(a+b = \frac{1}{9}(10a+b)\) → \(9(a+b)=10a+b\) → \(9a+9b=10a+b\) → \(8b=a\).
Karena \(a,b\) digit (0–9) dan \(a \geq 1\): \(b=1 \Rightarrow a=8\) → bilangan 81. Tidak ada nilai \(b \geq 2\) yang menghasilkan digit valid.
Only solution: \(b=1, a=8\).
Hanya 81 / Only 81
c)
Bilangan tiga digit \(\overline{abc}=100a+10b+c\), syarat: \(9(a+b+c)=100a+10b+c\) → \(91a+b=8c\).
Coba \(a=1\): \(91+b=8c\). Karena \(c \leq 9\): \(8c \leq 72 < 91\). Tidak ada solusi untuk \(a=1\).
Tidak ada bilangan tiga digit yang memenuhi.
No three-digit solution exists.
Tidak ada / None

Geometry & Counting
a)
Persegi \(k\times k\) dalam grid \(4\times4\): ada \((4-k+1)^2\) buah.
\(1\times1\): \(4^2=16\)  |  \(2\times2\): \(3^2=9\)  |  \(3\times3\): \(2^2=4\)  |  \(4\times4\): \(1^2=1\)
Total = \(16+9+4+1=30\)
30 persegi / 30 squares
b)
Total persegi panjang (termasuk persegi) dalam grid \(4\times4\) = \(\binom{5}{2}\times\binom{5}{2} = 10\times10=100\).
Persegi panjang bukan persegi = \(100-30=70\)
70 persegi panjang / 70 rectangles
c)
Grid \(5\times5\): \(5^2+4^2+3^2+2^2+1^2 = 25+16+9+4+1=55\)
55 persegi / 55 squares

Arithmetic / Aritmetika
a)
Persamaan 1: \(b + s = 30\) (semua dijawab)
Persamaan 2: \(4b - 2s = 54\)
b+s=30 dan 4b−2s=54
b)
Dari persamaan 1: \(s=30-b\). Substitusi ke 2: \(4b-2(30-b)=54\) → \(4b-60+2b=54\) → \(6b=114\) → \(b=19\)
19 soal benar / 19 correct
c)
Menjawab 25 soal: \(b+s=25\), \(4b-2s=54\).
\(s=25-b\) → \(4b-2(25-b)=54\) → \(6b=104\) → \(b=17{,}3...\) bukan bilangan bulat.
Periksa: skor 54 dengan 25 soal dijawab tidak mungkin menghasilkan bilangan bulat. Skor maksimum 25 soal = 100, minimum = −50. \(4b-2(25-b)=54\) → \(b=17{,}33\). Tidak ada solusi bulat.
No integer solution exists for this case — the score of 54 with exactly 25 answers is impossible.
Tidak mungkin / Impossible

a)
\(T_{12}=\frac{12\times13}{2}=\frac{156}{2}=78\)

b)
\(T_n+T_{n+1}=\frac{n(n+1)}{2}+\frac{(n+1)(n+2)}{2}=\frac{(n+1)(n+n+2)}{2}=\frac{(n+1)(2n+2)}{2}=\frac{2(n+1)^2}{2}=(n+1)^2\)
Karena hasilnya \((n+1)^2\), maka selalu merupakan bilangan kuadrat sempurna. ✓
Since the result is \((n+1)^2\), it is always a perfect square. ✓
Terbukti: \(T_n+T_{n+1}=(n+1)^2\)

\(T_{19}=\frac{19\times20}{2}=190\) (belum melebihi 200)
\(T_{20}=\frac{20\times21}{2}=210 > 200\) ✓
