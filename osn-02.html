Arithmetic / Aritmetika
a) Dua Persamaan / Two Equations
Persamaan 1: \(b + s = 30\) (semua soal dijawab / all answered)
Persamaan 2: \(4b - s = 75\) (sistem skor / scoring system)
b+s=30 dan/and 4b-s=75
b) Soal Benar / Correct Answers
Dari persamaan 1: \(s = 30-b\). Substitusi ke persamaan 2:
\(4b-(30-b)=75 \Rightarrow 5b=105 \Rightarrow b=21\), \(s=9\)
Cek: \(4(21)-1(9)=84-9=75\) ✓
21 soal benar / 21 correct
c) Peserta Kedua / Second Contestant
Menjawab 25 soal: \(b+s=25\), \(4b-s=75\).
Jumlahkan: \(5b=100 \Rightarrow b=20\), \(s=5\)
Cek: \(4(20)-1(5)=80-5=75\) ✓
20 soal benar / 20 correct

Number Theory / Teori Bilangan
a) Pasangan Prima Kembar / Twin Prime Pairs
(3,5), (5,7), (11,13), (17,19), (29,31), (41,43)
6 pasangan / 6 pairs
b) Jumlah Semua / Sum of All
3+5+5+7+11+13+17+19+29+31+41+43
= (3+5)+(5+7)+(11+13)+(17+19)+(29+31)+(41+43)
= 8+12+24+36+60+84 = 224
224
c) Pembuktian / Proof
Untuk prima \(p > 3\): semua prima > 3 berbentuk \(6k+1\) atau \(6k+5\) (ekuivalen \(6k-1\)).
Kasus 1: \(p=6k+1\) maka \(p+2=6k+3=3(2k+1)\) bukan prima kecuali \(k=0\). Untuk \(k\geq1\): \(p-1=6k\) habis dibagi 6. ✓
Kasus 2: \(p=6k-1\) maka \(p+2=6k+1\) (prima). Di sini \(p+1=6k\) habis dibagi 6. ✓
Jadi untuk setiap pasangan prima kembar \((p,p+2)\) dengan \(p>3\), selalu ada \(p-1\) atau \(p+1\) yang habis dibagi 6.
Terbukti / Proven ✓

Pattern & Proof / Pola & Pembuktian
a) Jumlah 1+3+5+...+19
Suku terakhir \(2n-1=19 \Rightarrow n=10\). Jumlah = \(10^2=100\)
100
b) Jumlah 1+3+5+...+99
Suku terakhir \(2n-1=99 \Rightarrow n=50\). Jumlah = \(50^2=2500\)
2500
c) Jumlah 21+23+...+99
\(21+23+\cdots+99 = (1+3+\cdots+99)-(1+3+\cdots+19)\)
\(= 2500-100=2400\)
Verifikasi: jumlah suku = \(\frac{99-21}{2}+1=40\). Rata-rata = \(\frac{21+99}{2}=60\). Total = \(40\times60=2400\) ✓
2400
